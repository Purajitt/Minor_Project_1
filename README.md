# 📊 WhatsApp Chat Analyzer (GroupDNA)

A Python-based Data Science Minor Project that analyzes WhatsApp group chat data to generate meaningful insights about group activity, user participation, media sharing, deleted messages, and communication patterns.

This project was developed using **Python**, **Pandas**, and **Google Colab** as part of my **Data Science Internship at Unlox Academy**.

---

## 📌 Project Overview

The objective of this project is to analyze exported WhatsApp chat data and transform raw messages into meaningful insights using data analysis techniques.

The project demonstrates how Python and Pandas can be used to clean, process, analyze, and summarize real-world text data.

---

## 🚀 Features

- 📂 Read and process WhatsApp chat data
- 🧹 Data cleaning and preprocessing
- ❌ Remove system-generated messages
- 📷 Analyze media messages
- 🗑️ Analyze deleted messages
- 👥 Participant-wise message count
- 📅 Busiest day analysis
- ⏰ Busiest hour analysis
- 💬 Most frequently used words
- 📊 Final GroupDNA Report
- 🧠 Personality archetype generation
